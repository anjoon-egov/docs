# POS Integration Configuration

## Introduction <a id="Introduction"></a>

Point of Sales \(POS\) machine, it is a machine that helps in handling transaction processing. This machine accepts and verifies the payments which are made by citizens for prevailing the services of DIGIT.

POS facilitates a middleware app developed in order to verify the payment process between the DIGIT module and the payment.

In this case, no data is required from the state team.
